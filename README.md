# 🤖 Generative AI Chatbot using Google Gemini API

## 📌 Project Overview

This project demonstrates how to build an AI-powered chatbot using Google's Gemini API and Python. It showcases prompt engineering, multimodal AI capabilities, streaming responses, chat history, image understanding, and token counting.

The notebook is designed as a portfolio-ready case study to demonstrate practical Generative AI development using the official Gemini Python SDK.

---

## 🎯 Objectives

- Learn Google Gemini API integration
- Build an AI-powered chatbot
- Understand prompt engineering
- Explore multimodal AI capabilities
- Generate intelligent responses using Large Language Models (LLMs)
